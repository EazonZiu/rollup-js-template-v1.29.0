# rollup-js-template-v1.29.0

#### master:

使用 rollup 的 javascript 模版

#### babel:

使用 rollup & babel 的 javascript 模版, 为兼容旧版本 javascript

#### ts:

使用 rollup 的 typescript 模版

#### 使用方式:

1.修改项目名称和其 package.json 中"name"的名称

2.执行命令

```bash
npm install
```

3.打包

```bash
npm run build
```

4.测试

```bash
npm run test
```
